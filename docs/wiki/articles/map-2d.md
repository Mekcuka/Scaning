# Карта 2D

Раздел **Карта** (`/map`) показывает объекты инфраструктуры и POI активного проекта на подложке OpenLayers.

## Панель «Слои»

Слева (или в боковой панели) — переключатели:

- **Подложка** — спутник, схема и т.д.
- **Группы подтипов** — дороги, трубопроводы, площадки и др.
- **POI** — точки интереса (скважины, площадки).
- **Радиусы** — зоны вокруг POI для анализа.

Настройки видимости слоёв **сохраняются в браузере** на проект (localStorage). Видимость импортированных источников из внешних систем — в базе данных (чекбоксы «Источники»).

## Объекты на карте

- **Точки** — клик открывает карточку объекта.
- **Линии** — трубопроводы, дороги; в режиме редактирования можно двигать вершины.
- **Полигоны** — площадные объекты.

При большом числе объектов карта подгружает данные по видимой области (bbox), чтобы не тормозить.

## POI

POI — ключевые точки для расчёта стоимости и матрицы. Выбор POI на карте влияет на контекст AI-помощника и вкладки «Потоки» / «Матрица».

## Режим редактирования

Роли с правом записи могут менять геометрию и метаданные объектов. Опасные массовые операции (удаление) требуют подтверждения.

## 3D

Переключение в 3D — отдельный режим (MapLibre + Three.js). Custom GLB загружаются через «Импорт 3D».
