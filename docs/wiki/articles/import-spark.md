# Импорт данных

Раздел **Импорт** (`/import`) доступен ролям **admin**, **analyst**, **data_manager**. Viewer импорт не выполняет.

## Поддерживаемые форматы

- CSV, GeoJSON, KML, Shapefile (zip)
- Подключения к внешним API
- **Проект Искра (SPARK)** — специализированный импорт с маппингом слоёв

## Типичный сценарий

1. Откройте `/import` и выберите тип импорта.
2. Укажите **целевой проект** (если требуется).
3. Загрузите файл или настройте подключение.
4. Дождитесь завершения — импорт может идти **фоновой задачей**.
5. Проверьте результат на **карте** и в **журнале импортов**.

## Искра (SPARK)

Импорт Искры переносит слои и объекты по правилам маппинга подтипов инфраструктуры. После импорта проверьте видимость слоёв в панели «Слои» на карте.

## Журнал

История импортов доступна через API и AI-tool `list_import_logs` — в UI смотрите статус последних операций на странице импорта.

## Ошибки

При ошибке в журнале будет код и сообщение. Частые причины: неверная проекция координат, неизвестный подтип объекта, нет прав на проект.
