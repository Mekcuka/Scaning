- [ ] Страница создания/редактирования матрицы
- [ ] Вкладки: Общее, Критерии, Альтернативы, Веса, Результаты
- [ ] Редактор критериев
- [ ] **Вертикальная таблица альтернатив**: строки = параметры (9 подтипов + инженерные параметры + стоимость + рейтинг), столбцы = сценарии

- [ ] Кнопка "Рассчитать" с прогрессом
- [ ] **Страница «Ставки»**: 16 показателей в тыс. ₽ (FR-4.1.2)
- [ ] **Загрузка инфраструктуры**: раздел «Импорт» + ручное нанесение на «Карте»
- [ ] **Матрица**: вертикальная таблица + переключатель карточного вида
- [ ] **Страница точек интереса**: 4 порога external + 4 max_total_line + 4 нормы км/КП + инженерные параметры *(спецификация готова)*
- [ ] **Страница анализа окружения**: таблица с 9 подтипами, статусами, ручным переопределением объектов

### Неделя 12: Frontend — Scenarios + One-pager
- [ ] **Базовый вариант**: автоматический расчёт по ближайшим объектам
- [ ] **Сценарии инженера**: создание сценария с ручными корректировками (переключение на строительство, выбор альтернативного объекта, корректировка стоимости)
- [ ] **Сравнение сценариев**: вертикальная матрица с несколькими столбцами
- [ ] **Одностраничник для руководства**: структура (заголовок, карта, финальный вариант, инженерные параметры, дорожная карта, рекомендация)
- [ ] **Экспорт PDF**: генерация печатной версии одностраничника
- [ ] **Экспорт PowerPoint (PPTX)**: генерация презентации для руководства
- [ ] **Дорожная карта**: визуализация этапов (Разведка → Изыскания → ПИР → Бурение → Строительство → Эксплуатация)

### Результат
- Полный цикл создания и расчёта матрицы
- Поддержка WSM и TOPSIS
- UI для управления критериями и альтернативами
- **Инфраструктура проекта**: 9 подтипов, анализ окружения, базовый вариант + сценарии
- **Одностраничник**: PDF + PPTX для руководства
