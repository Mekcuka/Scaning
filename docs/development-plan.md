## Этап 4: Матрица решений + Инфраструктура проекта (Недели 9-12)

### Неделя 9: Backend — Infrastructure variants
- [ ] **Модели инфраструктуры**: projects, points_of_interest, infrastructure_layers, infrastructure_objects, poi_infrastructure_analysis, implementation_variants, variant_infrastructure_items, variant_cost_overrides
- [ ] **9 подтипов инфраструктуры**: автодорога, нефтепровод, водопровод, ЛЭП, кустовые площадки; внешние: ГКС, ГТЭС, ПС, НПЗ

- [ ] **API расчёта стоимости**: расстояние × ставка + инженерное оборудование

### Неделя 10: Frontend — Matrix UI + Infrastructure
- [ ] **Вертикальная таблица альтернатив**: строки = параметры (9 подтипов + инженерные параметры + стоимость), столбцы = POI
- [ ] **Группы бейджей для инженерных параметров**: все доступные значения отображаются серым, выбранное подсвечивается цветом
- [ ] Фильтр POI с превышениями; выбор столбца → линии на мини-карте
- [ ] **Страница «Ставки»**: 16 показателей в тыс. ₽ (FR-4.1.2)
- [ ] **Матрица**: вертикальная таблица + переключатель карточного вида
- [ ] **Страница точек интереса**: 4 порога external + 4 max_total_line + 4 нормы км/КП + инженерные параметры
- [ ] **Страница анализа окружения**: таблица с 9 подтипами, статусами, ручным переопределением объектов

### Неделя 11: Frontend — Matrix + One-pager
- [ ] **Базовый расчёт**: автоматический расчёт по ближайшим объектам для каждой POI
- [ ] **Сравнение POI**: вертикальная матрица со столбцами = точки интереса
- [ ] **Одностраничник для руководства**: структура (заголовок, карта, финальный вариант, инженерные параметры, дорожная карта, рекомендация)
- [ ] **Экспорт PDF**: генерация печатной версии одностраничника
- [ ] **Экспорт PowerPoint (PPTX)**: генерация презентации для руководства
- [ ] **Дорожная карта**: визуализация этапов (Разведка → Изыскания → ПИР → Бурение → Строительство → Эксплуатация)

### Результат
- **Инфраструктура проекта**: 9 подтипов, анализ окружения, базовый расчёт по POI
- **Инфраструктурная матрица**: вертикальная таблица + карточный вид
- **Одностраничник**: PDF + PPTX для руководства
