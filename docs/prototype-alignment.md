## Параметры ввода: прототип vs документация

| Параметр / группа | Каталог `id` | Прототип | Целевой UI |
|-------------------|--------------|----------|------------|
| Название, описание проекта | `project_*` | частично | Проекты |
| 16 ставок (тыс. ₽) | `rate_*` | **Да** (`rates`) | Ставки |
| Пороги по умолчанию (8) | `threshold_*_km` | нет | Проект → настройки |
| POI: добыча, КП | `poi_planned_*`, `poi_production_per_well` | **Да** | Проект → POI |
| POI: флюид, закачка воды | `poi_fluid_type`, `poi_water_injection_volume` | **Да** (форма POI, матрица, отчёт) | Проект → POI |
| POI: подготовка нефти, сбор скважин | `eng_oil_preparation`, `eng_well_gathering` | после ревизии | 5 + 3 бейджа |
| POI: 5 инженерных | `eng_*` | 5 бейджей | без сепарации/ПС |
| POI: 9 порогов | `threshold_*_km` | нет | Аккордеон POI |
| Ранжирование: экспертные оценки | `rank_criterion_risk` и др. | статика | Таблица по сценариям |
| Ранжирование: веса, алгоритм | `rank_criterion_weights` | частично | Ранжирование |

## Реализовано в прототипе (mock)

### Карта — OpenLayers 9

- Библиотека: **OpenLayers** (`ol@9.2.4`, CDN), модуль `js/map-ol.js`.
- Подложки: OSM, Satellite, Terrain (FR-2.1.2).
- Панель слоёв: данные инфраструктуры, пороговые радиусы (вкл/выкл), выбор basemap.
- Иконки подтипов: **Lucide** (SVG overlay), не FontAwesome.
- Инструменты: выбор, точка, линия (FR-2.5.6); диалог ручного ввода — упрощённо.
- Линии подключения POI → объекты по **выбранному сценарию** (матрица / отчёт / главная карта).
- Мини-карты: боковая панель матрицы (`matrix-mini-map-container`), одностраничник (`report-map`).

### Импорт — отдельный раздел

- Раздел **«Импорт»** в боковом меню (не вкладка на экране «Карта»).
- Mock API (URL, реестр, auth, тест, прогресс), импорт файлов (dropzone, превью CSV), история последних 3 операций (`localStorage`: `dm-import-history`).
- С экрана «Карта» — ссылка «раздел Импорт»; из проекта (вкладка «Инфраструктура») — кнопка «Импорт».

### Ставки стоимости (FR-4.1.2)

- Отдельная страница **«Ставки»**: **16 показателей**, ввод в **тыс. ₽** (`js/cost-rates-spec.js`); без газопровода, пункта сбора, водозабора, морского порта (внешний подтип).
- Группы: линейные внутренние (4), площадные внешние (5), кустовые площадки (1), инженерное оборудование (7: электро, закачка, газ, 4× подготовка нефти).
- Сохранение в `localStorage`: `dm-cost-rates-{projectId}`; сброс к значениям по умолчанию.
- В карточке проекта — краткая сводка + переход на полную страницу.

### Матрица решений (FR-8)

- Единый источник данных: `MATRIX_SPEC` в `js/matrix-spec.js`.
- **Табличный вид** (по умолчанию): вертикальная матрица, строки = параметры, столбцы = сценарии (базовый + сценарии).
- **Карточный вид**: переключатель «Таблица / Карточки».
- Секции: внутренние (в т.ч. сбор скважин), инженерные параметры (бейджи), инженерное оборудование, внешние.
- Правила взаимоисключения инженерных параметров (`applyEngineeringRules`).
- Фильтр сценариев с превышениями; выбор столбца → линии на мини-карте.

### Прочее

- Ранжирование: Chart.js, TOPSIS (статические данные).
- Одностраничник: карта, таблица, дорожная карта, рекомендация; mock PDF/PPTX.
- Onboarding, тёмная/светлая тема, toast-уведомления.
