Планировщик Steiner — прототип

Клик по полю — добавить терминал. Колёсико — масштаб. Первый = start, последний = end. Перед расчётом запустите API: uvicorn network_planner.api:app --port 8080. Кнопки SteinerPy и GeoSteiner — расчёт через внешние солверы.

● start● end● intermediate ● steiner ● hub ● attach ● waypoint ◎ радиус — дерево ⊕ колёсико — zoom
Параметры для расчёта:

Идентификаторы вида {prefix}:0, {prefix}:1 …

Длина ребра терминал→{prefix}:0; 0 — hub в точке терминала.

0 — одно ребро между узлами; иначе промежуточные steiner:waypoint:*.

0 — без ограничений. Steiner-точки ближе R к терминалу отталкиваются на границу зоны (в обоих солверах). Hub/attach-точки всегда остаются у терминала.

Угол между ребром терминала и направлением на магистраль. 90° — перпендикуляр.

0 — выключено; 0.5–2 — мягкое предпочтение; 2–5 — жёсткое.

GeoSteiner — точный алгоритм (efst + bb). Дополнительных параметров нет: угол примыкания и его штраф не поддерживаются библиотекой. Используется общая часть настроек выше + пост-процессинг (радиус, лимиты, hub-узлы).
