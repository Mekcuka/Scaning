# AI-помощник

Иконка в **шапке** открывает панель чата. Помощник подключён к данным Atlas Grid через **tools** (не выдумывает цифры по проектам).

## Что умеет

- Список проектов, POI, объектов на карте, тарифы, статус задач.
- Анализ POI и схемы потоков — по данным из базы.
- **Справка по интерфейсу** — через внутреннюю wiki (`search_wiki`).

## Контекст UI

Помощник видит:

- активный **проект**;
- **выбранный POI** (если выделен на карте или в списке);
- текущую **страницу** (карта, матрица, потоки…).

Меньше уточняющих вопросов «какой проект?».

## Быстрые команды (chips)

Под полем ввода — кнопки вроде «Проекты», «Объекты на карте», «Справка по разделу». Нажатие подставляет типовой вопрос.

## Опасные действия

Запуск анализа, смена тарифов, удаление объектов — помощник предложит **подтвердить** действие перед выполнением.

## Размышления модели

Если модель поддерживает chain-of-thought, рассуждения свёрнуты под «Размышления модели» — основной ответ отдельно.

## Cursor MCP

Для разработчиков доступен HTTP MCP (`atlas-grid`) и wiki resources `wiki://*`. Настройка — в блоке «Подключение Cursor MCP» в панели помощника.
