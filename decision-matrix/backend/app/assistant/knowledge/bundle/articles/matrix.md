# Матрица решений

Раздел **Матрица** (`/{id-проекта}/matrix`) сравнивает **точки интереса (POI)** проекта по результатам пространственного и экономического анализа.

## Что показывает матрица

- Стоимость подключения к инфраструктуре для каждого POI.
- Ранжирование и цветовая индикация (лучше / хуже относительно порогов).
- Связь с **кандидатами** — вариантами привязки к объектам инфраструктуры.

## Перед использованием

1. Выберите **активный проект**.
2. Убедитесь, что POI созданы и на карте есть релевантная инфраструктура.
3. Запустите **анализ POI** (один POI или «анализ всех») — расчёт выполняется фоновой задачей.

Пока задача выполняется, статус виден в журнале задач (иконка в шапке).

## Интерпретация

Матрица не заменяет инженерное решение — она агрегирует расчётные показатели по заданным **тарифам** и **порогам расстояний** проекта. Изменение ставок на `/{id-проекта}/parameters/rates` пересчитывает экономику.

## AI-помощник

На вкладке матрицы можно спросить: «какой POI дешевле», «покажи анализ выбранного POI» — помощник вызовет tools с данными проекта, а не угадает цифры.
