# Параметры — земляные работы площадки

Раздел **Параметры → Земляные работы** (`/parameters/earthwork`) — таблица по **всем точечным объектам** проекта, для которых доступны земляные работы: ГКС, ПС, кусты, **карьер песка**, ДНС и т.д. **Не показывается:** только «Узел».

Подробнее про расчёт объёмов и DEM: [pad-earthwork-volumes.md](pad-earthwork-volumes.md).

## Где открыть

1. Выберите **активный проект** в шапке.
2. Боковое меню → **Параметры**.
3. Вкладка **Земляные работы**.

## Какие поля можно менять

| Колонка | Смысл |
|---------|--------|
| **Длина, м** | Габарит площадки по длине |
| **Ширина, м** | Габарит по ширине |
| **Высота насыпи, м** | Толщина слоя песка / насыпи (по умолчанию **1**) |
| **Опорная отметка, м** | Подошва площадки (ноль выемки на плоской отметке; по умолчанию **0**) |
| **Поворот / НДС, °** | Поворот прямоугольника; на кустах — азимут ряда скважин (0…360°) |

Сохранение — **при выходе из ячейки или Enter** (как на вкладке «Пропускная способность»). Пустое значение **очищает** параметр на объекте.

## Что остаётся на карте

На **Параметры → Земляные работы** нет:

- редактора **Схема…** (контур, полигон, **Генератор** скважин — только на кустах);
- загрузки **DEM** и кнопки **Рассчитать** (отсыпка / выемка);
- **Применить к песку**.

После изменения L/W/H на этой странице объёмы **не пересчитываются автоматически** — откройте объект на **Карте → Логистика** и нажмите **Рассчитать**, если нужны актуальные цифры.

См. также **Параметры → Точки подключения** — шаблон, куда подходят линии к контуру площадки в режиме «Площадки»: [parameters-footprint-connections.md](parameters-footprint-connections.md).
