# Логистика

Раздел **Логистика** (`/logistics/...`) объединяет расчёт логистики проппанта и табличное редактирование объёмов песка по объектам проекта.

## Вкладки

| Вкладка | Маршрут | Содержание |
|---------|---------|------------|
| Логистика | `/logistics/schematic` | Расчёт логистики песка, схемы подсетей, timeline |
| Объём песка | `/logistics/sand` | Массовое редактирование спроса (`sand_volume_m3`, план по годам) |

По умолчанию открывается вкладка **Логистика**.

## Расчёт логистики песка

На вкладке **Логистика** задайте **горизонт расчёта** (начало — автоматически, конец — с возможностью правки) и **срез для схемы** (`as_of`). Кнопка **«Рассчитать логистику песка»** запускает расчёт; результат **сохраняется в БД** проекта.

После расчёта для каждой подсети доступны таблицы карьеров и потребителей, интерактивная схема и слайдер года среза.

## Объём песка

Вкладка **Объём песка** — таблица точечных объектов с полем спроса (кроме узла и карьера). Excel-выгрузка поддерживается. Детальное редактирование плана по годам — на **карте**, вкладка **Логистика** в карточке объекта.

## Плечо возки

После расчёта таблица **«Плечо возки»** (карьер, объём, км) доступна: вкладка **Логистика** карточки на карте, столбец на **Логистика → Объём песка**, колонка в таблице потребителей на вкладке **Логистика**.

## Связь с картой и потоками

- Объёмы и даты ввода задаются на **карте**; для расчёта нужна построенная сеть автодорог с карьерами.
- **Потоки** (технология и экономика) — отдельный раздел по выбранному POI; логистика песка работает **по всему проекту**.
