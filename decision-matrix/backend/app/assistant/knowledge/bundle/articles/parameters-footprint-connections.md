# Параметры — точки подключения линий

Раздел **Параметры → Точки подключения** (`/parameters/footprint-connections`) задаёт **шаблон проекта**: с какой **стороны света** (С/Ю/В/З) или от **центра** площадки должны визуально подходить линии каждого типа (трубопровод, дорога, ЛЭП и т.д.).

Настройка работает в режиме карты **«Площадки»**: концы линий **отображаются** на выбранном ребре контура, но в базе координаты остаются в **центре** точечного объекта. Подробнее про контуры — [map-2d.md](map-2d.md), про габариты площадок — [parameters-earthwork.md](parameters-earthwork.md).

## Где открыть

1. Выберите **активный проект** в шапке.
2. Боковое меню → **Параметры**.
3. Вкладка **Точки подключения**.

В шапке страницы — счётчики «типов задано» и «объектов на карте», индикатор **Сохранено** / **Сохранение…**, кнопка **Открыть карту**.

## Шаблон подключений (левая колонка)

Для каждого **типа линии** (вкладки с цветным бейджем стороны):

| Элемент | Действие |
|---------|----------|
| Вкладки типов линий | Переключение активного типа; бейдж — сторона или «—» |
| Кнопки **С / З / В / Ю** | Быстрый выбор стороны света |
| **Центр** | Подключение к центру площадки (без смещения к ребру) |
| Выпадающий список | То же + **Не задано** (тип не участвует в шаблоне) |
| **Позиция вдоль стороны (0…1)** | Точка на ребре: 0 — начало ребра, 1 — конец, 0.5 — середина |
| **Сбросить** | Убрать настройку для текущего типа линии |
| **Схема шаблона** | Прямоугольник 120×80 м; клик по ребру или по маркеру в легенде задаёт сторону; клик по строке легенды переключает тип линии |

Изменения **сохраняются автоматически** на сервер (шаблон проекта). При первом открытии после обновления данные из браузера могут быть перенесены на сервер.

**Важно:** шаблон задаёт **стороны света**, а не номера рёбер. При применении к каждой площадке учитывается её **поворот** (`pad_rotation_deg` / схема): «восток» на одном объекте может попасть на другое ребро контура, чем на другом.

## Применить на карте (правая колонка)

1. Настройте хотя бы один тип линии в шаблоне.
2. При необходимости выберите **фильтр подтипа** (все earthwork-точки, только кусты, ГКС и т.д.).
3. Нажмите **Применить ко всем** или **Применить к подтипу** (вторая кнопка активна при выбранном подтипе).
4. Подтвердите в модальном окне — показывается сводка шаблона и число объектов.

Во время применения отображается **прогресс-бар**. Объекты **без контура** площадки пропускаются.

После успешного применения:

- **Отменить последнее (N)** — откат последней массовой операции на этой странице;
- на **карте** — **Ctrl+Z** (если перейти на карту до других правок) откатывает то же применение.

Режим просмотра (роль без записи): шаблон и схема доступны только для чтения.

## Точечная настройка на карте

В карточке earthwork-объекта (вкладка **Основное**, только при режиме **«Площадки»**):

- выбор типа линии, стороны света и `t`;
- кнопка **Применить шаблон проекта** — только для этого объекта;
- клик по ребру на мини-схеме или на карте.

Earthwork-eligible — все точечные подтипы, **кроме «Узел»** (как на вкладке «Земляные работы»).
