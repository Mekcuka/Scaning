# Навигация по приложению

После входа открывается **боковое меню** (сайдбар). Разделы зависят от роли пользователя.

## Шапка приложения

Заголовок текущей страницы (например «Проекты», «Матрица решений», «Импорт данных») отображается в **глобальной шапке** (`app-header`), между меню и иконками AI-помощника, журнала задач и темы. Кнопки действий страницы (режимы таблицы, экспорт, выбор POI) остаются **в теле страницы**, не в шапке.

## Проект и контекст

Большинство экранов работают в контексте **активного проекта**. Его **UUID в адресной строке** — часть URL:

```
http://localhost:5173/{id-проекта}/map
http://localhost:5173/{id-проекта}/matrix
http://localhost:5173/{id-проекта}/parameters/rates
```

При переходе по разделам сайдбара id проекта сохраняется. Смена проекта на дашборде или в списке проектов обновляет URL. Страница **Проекты → карточка проекта** (`/projects/{id}`) — настройки POI и анализ; быстрые ссылки «Карта», «Матрица» ведут на `/{id}/map`, `/{id}/matrix`.

**Без id проекта в URL:** вход (`/login`, `/register`), список проектов (`/projects`), карточка проекта (`/projects/{id}`), администрирование (`/admin/*`).

**Старые адреса** (`/map`, `/matrix`, `/parameters/...` без префикса) автоматически перенаправляются на `/{активный-проект}/...`.

## Основные разделы

| Раздел | Маршрут | Назначение |
|--------|---------|------------|
| Дашборд | `/{projectId}` | Обзор |
| Проекты | `/projects` | Список проектов, создание (admin/analyst) |
| Карта | `/{projectId}/map` | 2D / **Площадки** / 3D — объекты и POI |
| Кустование | `/{projectId}/pad-clustering` | Раскладка устьев, траектории, 3D куста |
| Параметры | `/{projectId}/parameters/*` | Пропускная способность, **земляные работы**, **точки подключения**, песок, даты, **ставки** |
| Потоки | `/{projectId}/flows/*` | Технологическая, экономическая схемы, логистика песка |
| Матрица | `/{projectId}/matrix` | Сравнение POI по стоимости и кандидатам |
| Отчёты | `/{projectId}/report` | One-pager и печать |
| **Данные** | `/{projectId}/data/*` | Импорт, экспорт, импорт 3D (подвкладки) |
| Администрирование | `/admin/*` | Пользователи, журнал задач, параметры AI (только admin) |

### Подвкладки «Данные»

| Вкладка | Маршрут | Кто видит |
|---------|---------|-----------|
| Импорт | `/{projectId}/data/import` | admin, analyst, data_manager |
| Экспорт | `/{projectId}/data/export` | все роли |
| Импорт 3D | `/{projectId}/data/import-3d` | admin (загрузка); admin и владелец проекта (назначение) |

Старые URL `/import`, `/export`, `/import-3d` перенаправляются на `/{projectId}/data/...`.

## Выход

Кнопка **Выход** — в нижней части сайдбара (иконка).

## AI-помощник

Иконка помощника в **шапке** открывает чат. Он учитывает активный проект и текущую страницу.

**Администратор:** вкладка **Администрирование → AI-помощник** (`/admin/assistant`) — статус LLM и Wiki RAG, probe подключения, временный override (chat и embeddings) без рестарта сервера, справка по провайдерам и Cursor MCP. Постоянные ключи и URL — в `app.env` на сервере.
