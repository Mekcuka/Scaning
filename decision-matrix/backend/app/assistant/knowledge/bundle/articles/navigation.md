# Навигация по приложению

После входа открывается **боковое меню** (сайдбар). Разделы зависят от роли пользователя.

## Основные разделы

| Раздел | Маршрут | Назначение |
|--------|---------|------------|
| Дашборд | `/` | Обзор |
| Проекты | `/projects` | Список проектов, создание (admin/analyst) |
| Карта | `/map` | 2D-карта объектов и POI |
| Параметры | `/parameters/*` | Пропускная способность, песок, даты, **ставки** |
| Потоки | `/flows/*` | Технологическая, экономическая схемы, логистика песка |
| Матрица | `/matrix` | Сравнение POI по стоимости и кандидатам |
| Отчёты | `/report` | One-pager и печать |
| Импорт | `/import` | Загрузка данных (admin, analyst, data_manager) |
| Экспорт | `/export` | Выгрузка координат и GeoJSON |
| Импорт 3D | `/import-3d` | Custom GLB: загрузка (admin), назначение и превью (admin / владелец проекта) |
| Администрирование | `/admin/*` | Пользователи, журнал задач, параметры AI (только admin) |

## Проект и контекст

Большинство экранов работают в контексте **активного проекта**. Выберите проект на дашборде или в списке проектов — данные карты, POI и расчётов привязаны к нему.

## Выход

Кнопка **Выход** — в нижней части сайдбара (иконка).

## AI-помощник

Иконка помощника в **шапке** открывает чат. Он учитывает активный проект и текущую страницу.

**Администратор:** вкладка **Администрирование → AI-помощник** (`/admin/assistant`) — статус LLM и Wiki RAG, probe подключения, временный override (chat и embeddings) без рестарта сервера, справка по провайдерам и Cursor MCP. Постоянные ключи и URL — в `app.env` на сервере.
