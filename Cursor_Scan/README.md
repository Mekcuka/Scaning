# HTML-прототип СППР (MVP)

Интерактивный прототип интерфейса системы поддержки принятия решений для нефтегазовой отрасли. Создан по документации в [`../docs/`](../docs/); сводка соответствия FR — [`../docs/prototype-alignment.md`](../docs/prototype-alignment.md).

## Запуск

Откройте `index.html` в браузере (двойной клик или локальный сервер):

```bash
cd Cursor_Scan
npx serve .
# или
python -m http.server 8080
```

Для входа: **«Демо-вход без регистрации»** или форма с любым email/паролем.

## Технологии прототипа

- **OpenLayers 9.2.4** (`js/map-ol.js`) — карта, маркеры Lucide, линии по сценарию
- **Lucide** — иконки UI и подтипов на карте
- **Chart.js** — TOPSIS и чувствительность
- **Inter / JetBrains Mono**, светлая и тёмная тема
- Демо-данные: `mock-data.js`, спецификации: `matrix-spec.js`, `cost-rates-spec.js`

## Функции

- SPA без бэкенда, переключение экранов в `app.js`
- Карта: OSM / Satellite / Terrain, слои и радиусы, координаты и масштаб
- Импорт: mock API/CSV, прогресс, история в `localStorage`
- Ставки: сохранение `dm-cost-rates-{projectId}`
- Матрица: таблица/карточки, фильтр превышений, выбор сценария → линии на мини-карте
- Отчёт: мини-карта с линиями выбранного варианта

## Ограничения

Прототип не подключён к API. Расчёты матрицы и TOPSIS — статические данные; ставки проекта пока не пересчитывают ячейки матрицы автоматически.
